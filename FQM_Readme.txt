
                                  FQM 5 Readme
________________________________________________________________________________


This mod uses the Stellar and Effects portions of the Multimedia Pack:

http://mmp.spaceempires.net/

To use the mod, you should install the packs into your root SE5\Pictures
folder.

The following packs are necessary:

Effects Pack 4 (5.3 MB)
Models Pack 3 (93.9 MB)
Planet Pack 3 (38.9 MB)
Planet Surface 2 (3.5 MB)
Systems 1 (676 Kb)
